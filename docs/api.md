# 公开 API

本文记录言序侧稳定入口。内部 JSON 字段保持英文，是为了与 Swift 原生宿主和跨语言工具可靠通信；公开给言序作者的 API 使用中文。

## 常量

- `版本`：当前库版本。
- `名称`：`言序麦金塔界面`。
- `原生宿主名`：`yanxu-macos-ui`。

## 应用

`应用（名称）` 创建 `麦金塔应用`。

常用方法：

- `设强调色（颜色）`
- `添窗口（窗口）`
- `添菜单（菜单）`
- `添菜单项（标题，事件名）`
- `设设置（视图）`
- `设文档型（值）`
- `当事件（处理器）`
- `描述（）`
- `JSON（）`

`JSON（）` 会校验应用至少包含一个窗口。

`运行（应用）` 通过 `标准:原生` 装载已校验的 ABI v2 宿主，并打开真实麦金塔窗口。调用方顶层项目需要允许 `原生扩展 = true`。`更新（应用）` 只能在运行期事件处理器中调用，用新应用快照更新现有窗口。

## 窗口

`窗口（标题，根视图）` 创建 `麦金塔窗口`。

常用方法：

- `尺寸（宽，高）`
- `最小尺寸（宽，高）`
- `设可调整大小（值）`
- `设默认命令（事件名）`
- `添工具项（标识，标题，事件名）`
- `描述（）`

尺寸必须在 `0..16384` 范围内。

## 菜单与命令

`菜单（标题）` 创建 `麦金塔菜单`。

`命令（标题，动作）` 创建 `麦金塔命令`。`动作` 可以是 `麦金塔动作`；0.3 的字符串事件名继续兼容。

命令方法：

- `设快捷键（键，修饰键）`
- `设角色（角色）`

事件名必须是稳定标识，推荐使用 `领域.动作`，例如 `project.open`、`draft.save`。

## 对象化视图

视图工厂返回具体类，而不是统一返回属性容器。主要继承关系：

- `麦金塔视图`：共同树结构、修饰和序列化行为；
- `麦金塔文本视图`：提供 `设内容`；
- `麦金塔交互视图`：提供 `当触发`；
- `麦金塔按钮视图`：提供 `设标题` 和 `当点击`；
- `麦金塔输入视图`：提供 `当变化`；
- `麦金塔文本框视图`、`麦金塔多行文本视图`、`麦金塔开关视图`：提供类型匹配的 `设值` 或 `当文本变化`；
- `麦金塔容器视图`：提供 `添视图` 和 `设间距`；
- `麦金塔自定义视图`：承载任意 `kind` 的低层兼容扩展。

通用方法：

- `添修饰器（修饰器）`
- `内边距（大小）`
- `字体（名称，字重）`
- `无障碍标签（标签）`
- `框架（宽，高）`
- `帮助（说明）`
- `禁用（值）`
- `添子（视图）`
- `描述（）`
- `JSON（）`

`设（名称，值）`、`设样式（名称，值）`、`设事件（事件名）` 继续保留，用于兼容 0.3 和尚未对象化的宿主扩展。常规业务代码应优先使用具体类方法。

修饰器对象：

- `内边距（大小）` 创建 `麦金塔内边距修饰器`；
- `字体（名称，字重）` 创建 `麦金塔字体修饰器`；
- `视图框架（宽，高）` 创建 `麦金塔框架修饰器`。

修饰器可以通过 `视图.添修饰器（修饰器）` 组合；视图上的同名便捷方法创建并添加对应对象。

控件工厂：

- `文本（内容）`
- `标题（内容）`
- `按钮（标题，事件名）`
- `文本框（标识，占位，值）`
- `安全文本框（标识，占位）`
- `多行文本（标识，值）`
- `开关（标识，标题，值）`
- `图片（系统名）`
- `列表（标识，项目）`
- `进度（值）`

布局工厂：

- `垂直（子视图）`
- `水平（子视图）`
- `滚动（子视图）`
- `分组（子视图）`
- `表单（子视图）`
- `分栏（边栏，内容，详情）`
- `标签页（项目）`
- `分隔线（）`
- `间隔（大小）`

## 动作与事件

`动作（名称）` 创建 `麦金塔动作`。动作表示应用中的稳定用户意图，可同时交给按钮、菜单、工具栏和输入控件。其名称推荐采用 `领域.动作`，例如 `project.open`、`draft.save`。

`事件（名称，载荷）` 创建 `麦金塔事件` 对象。事件名采用同一套标识校验规则，载荷为字典。

动作是发出事件前的配置对象，事件是原生宿主回传后的运行时对象。`应用.当事件（处理器）` 注册唯一应用级处理器。`运行` 会把该处理器作为 ABI v2 持久回调传给原生宿主；按钮、菜单、工具栏和已注册动作的输入控件都进入这里。输入事件载荷为 `{「source」：标识，「value」：当前值}`。

处理器执行在言序 VM 所有者线程。原生层先投递到有界事件队列，再调用宿主事件泵，不会从任意 AppKit 后台线程直接重入言序。
