# 公开 API

本文记录言序侧稳定入口。内部 JSON 字段保持英文，是为了与 Swift 原生宿主和跨语言工具可靠通信；公开给言序作者的 API 使用中文。

## 常量

- `版本`：当前库版本。
- `名称`：`言序麦金塔界面`。
- `原生宿主名`：`YanxuMacUIHost`。

## 应用

`应用（名称）` 创建 `麦金塔应用`。

常用方法：

- `设强调色（颜色）`
- `添窗口（窗口）`
- `添菜单（菜单）`
- `添菜单项（标题，事件名）`
- `设设置（视图）`
- `设文档型（值）`
- `当事件（处理器）`
- `描述（）`
- `JSON（）`

`JSON（）` 会校验应用至少包含一个窗口。

`运行（应用）` 通过 `标准:原生` 装载已校验的 ABI v2 宿主，并打开真实麦金塔窗口。调用方顶层项目需要允许 `原生扩展 = true`。`更新（应用）` 只能在运行期事件处理器中调用，用新应用快照更新现有窗口。

## 窗口

`窗口（标题，根视图）` 创建 `麦金塔窗口`。

常用方法：

- `尺寸（宽，高）`
- `最小尺寸（宽，高）`
- `设可调整大小（值）`
- `设默认命令（事件名）`
- `添工具项（标识，标题，事件名）`
- `描述（）`

尺寸必须在 `0..16384` 范围内。

## 菜单与命令

`菜单（标题）` 创建 `麦金塔菜单`。

`命令（标题，事件名）` 创建 `麦金塔命令`。

命令方法：

- `设快捷键（键，修饰键）`
- `设角色（角色）`

事件名必须是稳定标识，推荐使用 `领域.动作`，例如 `project.open`、`draft.save`。

- `设事件（事件名）`
- `添子（视图）`

- `按钮（标题，事件名）`
- `文本框（标识，占位，值）`
- `安全文本框（标识，占位）`

## 事件

`事件（名称，载荷）` 创建 `麦金塔事件` 对象。事件名采用同一套标识校验规则，载荷为字典。

`应用.当事件（处理器）` 注册唯一应用级处理器。`运行` 会把该处理器作为 ABI v2 持久回调传给原生宿主；按钮、菜单、工具栏和已调用 `设事件` 的输入控件都进入这里。输入事件载荷为 `{「source」：标识，「value」：当前值}`。

处理器执行在言序 VM 所有者线程。原生层先投递到有界事件队列，再调用宿主事件泵，不会从任意 AppKit 后台线程直接重入言序。
