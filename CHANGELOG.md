# 更新记录

## 0.2.0 - 2026-07-15

- 将 `运行（应用）` 切换为言序 1.1.7 `标准:原生` + ABI v2 调用，不再启动外部进程。
- Swift 宿主新增 `yanxu_native_module_v2`，保留 ABI v1 入口以兼容旧验证工具。
- 清单升级到 `ABI = 2`，记录麦金塔 arm64 制品大小与 SHA-256。
- 文档更新为顶层应用授权 `原生扩展 = true` 的正式运行方式。

## 0.1.5 - 2026-07-15

- 随包发布麦金塔 arm64 原生动态库制品，并在清单中声明 SHA-256 校验和。
- 将文档更新为“应用作者只写言序代码，普通安装不需要手动配置 runner PATH”。
- 保留 runner 作为宿主开发和 1.1.x 过渡预览工具。

## 0.1.4 - 2026-07-15

- 新增 `运行（应用）`，可从言序直接启动 runner 并打开麦金塔窗口。
- 新增 `设运行器（路径）` 和 `YANXU_MACOS_UI_RUNNER` 支持。
- runner 采用脱离启动模式，言序程序无需等待窗口关闭。
- 示例改为直接调用 `界面.运行（应用）`。

## 0.1.3 - 2026-07-15

- 新增 `yanxu-macos-ui-runner` 本地预览宿主，可读取 `应用.JSON（）` 输出并打开真实麦金塔窗口。
- 文档明确说明示例输出 JSON 是应用描述，窗口由原生宿主/runner 呈现。

## 0.1.2 - 2026-07-15

- 修正 JSON 序列化 API 名称：使用标准库导出的 `JSON.序列化`，替代不存在的 `JSON.串化`。

## 0.1.1 - 2026-07-15

- 修正清单：移除尚未被当前言包接受的 `[[原生]]` 声明，保持 `yanbao add macos-ui` 可安装。
- 将原生宿主说明调整为源码随仓库提供，正式动态库制品待后续按平台与 SHA-256 校验发布。

## 0.1.0 - 2026-07-15

- 新增麦金塔原生界面描述树、窗口、应用和事件模型。
- 提供常用 SwiftUI 控件工厂：文本、按钮、文本框、开关、列表、表单、滚动区域、分组和布局容器。
- 新增 Swift Package 原生宿主骨架，使用 SwiftUI 渲染视图并由 AppKit 管理窗口。
- 加入示例、测试、API 文档和安全说明。
